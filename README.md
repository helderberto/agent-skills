# Agent Skills

Personal collection of Claude Code agent skills.

## Structure

Each skill is a directory containing a `SKILL.md` file with instructions for Claude Code.

## Available Skills

- **amend-push** - Amend last commit and force push
- **audit-deps** - Check dependencies for vulnerabilities
- **commit** - Create git commits following repo style
- **coverage** - Check test coverage for unstaged changes
- **create-pr** - Create GitHub pull requests
- **lint** - Run linting and formatting checks
- **open-pr** - Open current PR in browser
- **open-repo** - Open repository in browser
- **review-pr** - Review pull requests
- **safe-repo** - Check for sensitive data in repository
- **ship** - Stage all changes, commit, and push
- **test** - Run test suite

## Usage

Configure Claude Code to load skills from this repository.
